WebSocket Chatbox example



  • Websocket is standard in all modern browsers
  • So not any javascript library needed
  • Undertow supports websockets and rest
  • Rest interface is defined in Apache Kafka
  • Implementation of Rest interface is needed to communicate with Apache Kafka Streaming Applications
  • Implementation possibilities: Jetty,Undertow and Tomcat embedded.
  • Confluent uses Jetty
  • We prefer Undertow (Wildfly JBoss kernel)
  • We add the websocket possibility of undertow to make maximum use of the streaming data of Apache Kafka.
  • In production for security: use https instead of http protocol, and wss instead of ws protocol
